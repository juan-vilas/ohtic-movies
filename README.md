# Oh Movies!

![screenshot](./app-screenshot.png)

Find trending movies and tv shows and save them to your watchlist!

**Features:**

- See trending movies and tv shows
- Rating
- Release date
- Trailers
- Watchlist

# 🚀 Tech Stack

- ⚛ Expo
- 📂 File based navigation
- 📦 Redux Persistance
- 🛜 API
- 📝 Form validation
- 💅 Visually attractive

## Run on Android and IOS

### Android

`npm run android`

### IOS

`npm run ios`
